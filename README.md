# A Small Server  

## Start the server  
type **$./tiny port(an integer)** in command line   
```$./tiny 1234```  

## Access the server  
open your browser and fill in: **localhost:port**  
```127.0.0.1:1234/```


## Note  
1. static file stored in resource  
2. dynamic file stored in resource/cgi-bin/
